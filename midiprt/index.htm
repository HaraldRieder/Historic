Mad Harry's MIDI File Printer

The program makes most fun with 256 or more color displays and a color printer. To get an overview over the features look into the manual.

V4.1a for TOS, 2005 December

  • should run on all TOS systems: Atari ST(E)/TT/Falcon, Hades, Milan, Magic!, N.AES, Aranym, ...
  • needs GDOS: Speedo, NVDI or something similar.

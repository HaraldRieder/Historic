Mad Harry's MIDI File Printer

Leafing

These keys apply to the current document window:

Home:
to the first page on Atari TOS. To the top of the current page on Windows.
+Shift or Control:
to the last page on Atari TOS. To the first page on Windows.
End:
to the bottom of the current page on Windows.
+Shift or Control:
to the last page on Windows.
Arrow keys:
a certain amount of pixels up/down/left/right.
+Shift or Control:
horizontal/vertical slider up/down/left/right.

Mad Harry's MIDI File Printer

4.3 Speedo GDOS on Atari TOS

Known Speedo GDOS 4.x and 5.0 problems with HP Deskjet driver:

  • The quality must NOT be set to "draft". Otherwise lines will be missing.
  • Using text effect "light" leads to missing letters, therefore this attribute is no more supported by the MIDI file printer.
  • Using text effect "outlined" leads to a crash, this attribute is no more supported.

Set the print area switch to "off" and select the right paper size.

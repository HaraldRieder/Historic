Mad Harry's MIDI File Printer

1.3 What for and how ?

Learning the common Western music notation requires a lot of effort. The reason for that is, that this notation is much more complicated as it needs to be. If you can read the common music notation, you are probably proud of it, because it was hard to learn it.

The notation of the MIDI file printer is easy-to-learn. It is not at all complicated. It does NOT require:

  • different clef types for bass and treble
  • sharps and flats
  • necks
  • signs for rests
  • different variants to express the length of a 3/8 note
  • ...

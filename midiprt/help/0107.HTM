Mad Harry's MIDI File Printer

1.7 Delivery

You should have the following files:

MIDIPRT.EXE:
program variant for 32 bit Windows.
SCHMEDIT.EXE:
separate program to edit color schemes for 32 bit Windows.
*.ICO:
icon files for the Windows variant.
*.BMP:
background bitmap for the about me dialog.
MIDIPRTS.APP:
for ST-like Ataris and all others.
MIDIPRTT.APP:
requires at least 68020 processor and FPU (Atari TT-like).
*.AM:
ACS modules used by the Atari program variants.
SCHEMES\*:
a folder with *.PAR color schemes.
EXAMPLES\*:
a folder with free MIDI files and profiles.
HELP\INDEX.HTM:
this documentation as HTML.
MIDI_PRT.TXT:
this documentation as plain text.

You might have these files:

DESKTOP.CFG:
automatically generated, stores global ACS settings (Atari only).
EDITOR.CFG:
automatically generated, stores ACS editor settings (Atari only).
MIDI_PRT.CFG:
automatically generated, stores global settings.
MIDI_PRT.MI$:
default profile, generated by user action.

Mad Harry's MIDI File Printer

2.1 General

The MIDI File Printer may look different in different environments (e.g. single-/multitasking) and on different operating systems. Thus not all functions described below might be available in your current environment.

Since the Atari TOS variant it is an ACS application, you can load ACS modules with the usual mechanism. ACS modules you can find on http://acspro.atari.org . When Bubble Help is installed, most controls report their function by pressing the right mouse keys upon them.

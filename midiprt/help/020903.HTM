Mad Harry's MIDI File Printer

Edit Fields

These hot keys are available on Atari TOS whenever a text edit field is current:

Delete:
Erase selected text.
Insert:
Insert character from popup menu.
Control+X:
Cut selected text.
Control+C:
Copy selected text.
Control+V:
Paste from clipboard.
Arrow keys:
left/right.
+Shift:
to the beginning/end.
+Control:
one word left/right.

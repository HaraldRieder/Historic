Mad Harry's MIDI File Printer

3.2 Description

There are 4 sub-folders in the SCHEMES folder:

  1. 002_COLS: black & white shemes
  2. 016_COLS: at least 16 colors required
  3. 256_COLS: at least 256 colors required
  4. MANYCOLS: at least 32768 colors recommended

There are schemes representing pythagorean tunings, clean natural tunings, meantone tunings, chromatic tunings and piano keyboards.

For the chromatic schemes the key is not really critical. For the other ones you must transpose the key until it fits to the key of your piece of music. Use the transpose control of the parameter window to achieve this. For example, if you have a piece in 8 (G#) major you must transpose by 8 to make the scheme suitable for your piece.

When the MIDI file printer detects a key signature in the MIDI file it initializes the transpose control correspondingly.

You may find these abbreviations in the scheme file names:

75:
traditional piano keyboard (7+5)
66:
chromatic piano keyboard (6+6)
PYT:
pythagorean tuning
CLN:
clean natural tuning
MNT:
meantone tuning
CHR:
chromatic tuning

For PYT and CLN, the background and dodecime colors and patterns should harmonize with the note colors of the tonic, dominant, sub-dominant and other related notes. Notes with far relationships, e.g. a F# (6) in a piece written in C (0), should look contrasting. Read the file TUNINGS.HTM to learn more about tunings.

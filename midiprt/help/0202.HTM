Mad Harry's MIDI File Printer

2.2 The Desktop

On singletasking Atari TOS, and only there, the MIDI File Printer comes with an own desktop.

At any time there are at least 2 icons on the desktop:

  • New: a double-click to NEW opens the file selector and you can select one (or more, depending on the capabilites of your file selector) MIDI standard file(s) to be read.
  • Trash: close a MIDI file by dragging its icon onto the trash bin. Unload an ACS module with the same action.

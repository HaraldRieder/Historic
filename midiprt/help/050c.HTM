Mad Harry's MIDI File Printer

5.12 Version 4.1 for TOS and Windows (2005 June)

Corrected Faults:

  • (hollow) ellipses could be drawn partly outside the staff and not cut
  • more than 127 sub-bars could be adjusted (who needs this ?) but were not drawn correctly

New Features:

The Windows variant was re-implemented based on wxWindows version 2.4.2 (now called wxWidgets). This allows to publish the complete source code and to build Unix and Mac OS variants with minor code changes. The Windows and TOS variants now again provide the same feature set. The former Windows version 0.0 was based on a commercial library.

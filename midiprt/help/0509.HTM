Mad Harry's MIDI File Printer

5.9 Version 3.0 (1999 February)

Corrected Faults:

  • user interface harmonized a little bit with current fashion
  • page numbers start with 1, not 0 (a matter of philisophy and psychology)

New Features:

  • primitive help function implemented
  • open MIDI files are remembered
  • new triangle body for notes
  • new circular ends for notes
  • 256 colors supported
  • color and style for staff background selectable
  • colors and styles for each of the 12 note ends separately selectable
  • powerful controls for color and style manipulation
  • footline showing date, file name and size, page and total number of pages, and more ...
  • title line only on first page to save paper
  • sophisticated *.par examples, supporting pythagorean, natural and equal tunings

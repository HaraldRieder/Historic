Mad Harry's MIDI Enhancer 

V2.2 for Atari TOS

  • runs on all TOS systems: Atari ST(E)/TT/Falcon, Hades, Milan, Aranym, ...
  • tested on TOS 4.04 and Magic 6 on Falcon 030 and on ST Emulator Gold 1.6 with Windows 98, 2000 and XP
  • needs MIDI in & out interfaces (currently not supported by ST Emulator).
  • requires a multimode-capable MIDI synthesizer/expander with at least 16 voices or a Roland GS compatible instrument.

Here are:

  • the zip file for download (with manual in ASCII format)
  • the manual in HTML format

V2.2 for Palm OS

  • runs on Palm OS version 3.5 or higher.
  • tested on Palm OS 3.5 and Palm m105.
  • needs MIDI in & out at the serial interface or the RS-232 part of the universal connector (several assembly instructions in the WWW or below).

For installation install the file MIDIhanc.prc with the Palm Installer.

Mad Harry's MIDI Enhancer

1.4 Delivery

You should have the following files:

MIDIHANC.APP:
the program.
*.AM:
ACS modules for adjusting 3D look, dither mode, ...
TABLES\*:
tuner tables you can import into the tuner.
MIDIHANC.TXT:
this documentation.

You might have these files:

DESKTOP.CFG:
automatically generated, stores global ACS settings.
EDITOR.CFG:
automatically generated, stores ACS editor settings.
MIDIHANC.PAR:
automatically generated, stores main settings.
MIDIHANC.VEL:
automatically generated, stores velocity mappings.
MIDIHANC.PC:
automatically generated, stores program change mappings.

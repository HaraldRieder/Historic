Mad Harry's MIDI Enhancer

2.6 The Enter/Exit Strings Window

Enter MIDI strings in hexadecimal notation here. The "Enter" string will be sent when you press "start" in the Controller window, the "Exit" string will be sent when "start" is deselected.

Press the "Enter" and "Exit" buttons to test your MIDI strings and send them to the MIDI output of your computer.

There are 2 sets of strings. The one set is sent in pitch bend mode, and the other one is sent in scale tune mode.

Example purpose: With my Roland JV-1010 I use the "Enter" string "b9 00 50 20 00 c9 1f" in pitch bend mode to select a special performance. This user performance with index #31 can be selected by sending bank select and program change messages on the performance control channel #9. It has 16 identical piano sounds with a pitch bend range of "one half tone" per direction. The string above could be preceeded by the system exclusive message "f0 41 10 6a 12 00 00 00 00 00 00 f7" to guarantee that the JV-1010 is in performance mode before the performance is selected.

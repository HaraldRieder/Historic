Mad Harry's MIDI Enhancer

2.2 General

The MIDI Enhancer may look different in different environments (e.g. singletasking and multitasking). Thus not all functions described below might be available in your current environment.

The MIDI Enhancer saves all important settings on exit into

MIDIHANC.PAR:
controller, tuner, enter/exit strings
MIDIHANC.VEL:
volumes settings
MIDIHANC.PC:
program change mapping

Since it is an ACS application, you can load ACS modules with the usual mechanism (in multitasking mode only module loading on program start is supported). ACS modules you can find on http://acspro.atari.org and on my site.
